# Swagger-Himotoki
Simple script to generate Himotoki code in Swift 3.0 by Swagger yaml.


## Requirement
* python 2.7

## Install
```
pip install git+https://github.com/gitrocc/Swagger-Himotoki.git
```

## Usage
```
swagger_himotoki your_swagger.yaml --prefix <Class Name Prefix>
```
